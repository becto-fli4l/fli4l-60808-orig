fli4l - flexible internet router for linux             liezmoi.txt __FLI4LVER__
-------------------------------------------------------------------------------

La documentation française est disponible en deux versions, une
documentation pour chaque paquetage et une documentation pour l'ensemble
des paquetages. Deux formats sont mis à disposition, HTML et PDF.

La documentation complète est disponible dans un paquetage séparé sur
le site fli4l, après le téléchargement et l'installation de ce paquetage,
vous trouverez dans ces répertoires la documentation complète :

    - PDF :  doc/french/pdf/fli4l-__FLI4LVER__.pdf
    - HTML : doc/french/html/fli4l-__FLI4LVER__/index.html

La documentation individuelle pour le paquetage base est disponible
dans les répertoires :

    - PDF:  doc/french/pdf/base.pdf
    - HTML: doc/french/html/base/index.html

Pour les paquetages-Opt (<opt> sera remplacé par le nom du paquetage-Opt) :

    - PDF:  doc/french/pdf/<opt>.pdf
    - HTML: doc/french/html/<opt>.html/index.html

Pour information :

Le paquetage complet (à l'exception d'une partie des programmes du répertoire
windows) est soumis à la licence GPL (voir le fichier COPYING.GPL),
ces fichiers peuvent être distribués dans ça forme complète.

Les programmes dans le répertoire "windows", ne sont pas soumis à la
licence GPL, ils sont freeware. Dans le sous-répertoire imonc, vous pouvez
consulter le fichier eula-deutsch.txt pour la licence de ce programme.
