Information:
------------
Due to the changes in the build process you can find the generated fli4l
images now in the directory <your-fli4l-directory>/build.

In order to change this directory, please read the description about the build
process in the documentation.


Information:
------------
Augrund der Änderung des Prozesses zum Erzeugen der fli4l-Image-Dateien hat
sich auch das Verzeichnis für die fli4l-Dateien geändert. Als
Standardverzeichnis wird nun <dein-fli4l-verzeichnis>/build verwendet.

Um dieses Verzeichnis zu verändern, ist das Lesen der Dokumentation zu
empfehlen.
