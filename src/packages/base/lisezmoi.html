Documentation des paquetages OPT

Vous trouverez la documentation de chaque paquetage, dans le répertoire doc/french/pdf au format PDF et dans le répertoire doc/french/html au format HTML. Les noms des fichiers PDF sont simples à ouvrir <nom de l'Opt>.pdf, pour le format HTML il faut activer le fichier <nom de l'Opt>/index.html.

Documentation complète

Il existe également une autre version de la documentation, qui décrit l'ensemble des paquetages dans un seul document. Vous trouverez ce paquetage sur le site (www.fli4l.de). Après l'extration des fichiers, vous pourrez voir une nouvelle mise en page de cette documentation. Vous pouvez aussi consulter la documentation complète, directement sur le site (www.fli4l.de).
